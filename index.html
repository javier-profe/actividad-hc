Actividad 3.2: Usando las hojas de cálculo

En esta tarea tendrás que manipular y analizar los datos de una hoja de cálculo. Este tipo de manipulación de datos se hace con frecuencia: tendrás que dar formato a las celdas, utilizar funciones así como insertar gráficos que permitan mostrar de una manera distinta los datos de una hoja de cálculo. 

Este tipo de análisis es muy útil ya que te ayudará a interpretar la información y te permitirá una mejor toma de decisiones. 

Imagen alojada en Piqsels. Licencia CC0

Esta práctica está integrada dentro del conjunto de tareas/prácticas de la materia. 

En este caso, tendrás que practicar manipulando los datos de una hoja de cálculo de manera que te permita aprender a utilizar esta herramienta.
